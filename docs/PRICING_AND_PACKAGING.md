# Pricing and Packaging Decision (Draft)

## Packaging
1. Evaluation package:
   1. Native binary + header + smoke/benchmark scripts.
   2. Limited support SLA.
2. Production package:
   1. Versioned release artifacts and compatibility matrix.
   2. Priority support and regression-response commitment.

## Pricing Model
1. Evaluation: fixed-term pilot fee or waived for strategic partners.
2. Production: annual platform license with support tier add-on.
3. Expansion: additional environment/site fee where applicable.

## Commercial Guardrails
1. Claims must reference benchmark protocol + artifact evidence.
2. Security/legal docs must be finalized before external distribution.
3. Release gate sign-off required for production candidates.

## Approval State
1. Status: draft for GTM/Product review.
2. Decision owner: GTM Lead + Product Lead.
